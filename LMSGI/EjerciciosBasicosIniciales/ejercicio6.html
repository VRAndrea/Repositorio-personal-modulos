Encabezados jerárquicos

El encabezado principal

Este encabezado usa la etiqueta h1 y es para el título principal del documento porque es el nivel más alto en la jerarquía de encabezados

Encabezado nivel 2

Este encabezado usa la etiqueta h2 y se usa como el segundo nivel más alto después del h1. Encapsula grandes bloques de contenido que desarrollan lo contenido en h1.


Encabezado nivel 3

Este encabezado usa la etiqueta h3 y se usa para dividir en subtemas el contenido de lo de h2.


Encabezado nivel 4

Este encabezado usa la etiqueta h4 y se usa para dar detalles específicos dentro de los apartados de h3.


Encabezado nivel 5

Este encabezado usa la etiqueta h5 y se usa para desarrollar contenido más complejo que esté dentro del nivel 4 o h4.


Encabezado nivel 6

Este encabezado usa la etiqueta h6 y es el nivel más bajo de todos. Se usa poco cuando se necesita realizar una nota o inciso dentro del nivel h5.
